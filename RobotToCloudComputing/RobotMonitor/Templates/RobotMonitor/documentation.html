Main Page

This is the landing page where all of the currently supported robots are located. To view information about a robot click on its associated button.

Robot Menu Page

Once a robot has been selected you will be taken to its menu page. Here all available data is listed by it's time stamp. To see information for that time stamp click on that time and you will be taken to another page reffered to here as the display page. Here you will find data about the robot at that time stamp. This menu page also constains documentation regarding for the robot.

Display Page

On this page you will find:

  • Graphs containing information sent from the robot
  • Buttons to change what information will be displayed
  • Download button for the csv file containing the data for the selected timestamp
